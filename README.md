# 실시간 MMO 서버

### 사용 기술
- **언어**: C++
- **데이터 베이스**: Mysql
- **네트워크 라이브러리**: TCP, IOCP
- **데이터 직렬화**: Protobuf
- **utf변환**: boost-locale
- **운영 체제**: Windows
